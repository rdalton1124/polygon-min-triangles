Purpose

The goal of this project is to create a regular polygon using the minium number of triangles. This would be helpful in a 3-D environment where you want to make polygons for the sides of a specific object. This is advantegous over the 'standard' algorithm of repeatedly creating triangle with a corner, adjacent corner and the center because

  • There are less triangles. A n-sided polygon can be made with n - 2 triangles.
  • There is no need to keep track of the center. This especially useful if you want the angles to be on a sphere but the center is not necessarily on said sphere

Algorithm

The general alogrithm is as follows.

  • Start with an array of corner positions.
  • Start at a corner C.
  • Create a triangle out of C - C + 1 - C + 2 where C + 1 is the corner directly to the right of C and so on.
  • Remove C + 1 from the array.
  • If there are more than 3 corners left, call the algorithm with the remaining
